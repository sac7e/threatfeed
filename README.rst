================================================
ThreatFeed: a CLI tool to retrieve threat feeds
================================================

APIs:
======
- Safe Browsing APIs
- VirusTotal APIs
- DShield APIs
- PhishTank APIs

Installation:
=============

.. code-block:: bash

  $ pip install threatfeed

Usage:
======

.. code-block:: bash
  
  $ threatfeed query --type=hash ARG1 ARG2 ...

see also ``threatfeed --help``.
